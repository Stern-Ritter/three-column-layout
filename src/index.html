  • Используем свойства position: absolute и transform: translate.

    Для горизонтального и вертикального центрирования абсолютно позиционированного элемента сначала нужно задать свойства для его родителя: position: relative и min-height. А для центрируемого элемента, обязательно нужно указать position: absolute; и transform: translate(-50%).

  • Выравнивание по горизонтальному центру блочных (block) элементов с помощью margin: auto.

    В том случае, если нам нужно центрировать блочный элемент относительно страницы, то тут тоже есть давно работающий метод, который предполагает указание ширины элемента и margin: auto.

  • Использование flexbox для горизонтального и вертикального центрирования элементов.

    Свойства Flexbox-модели позволяют выровнять элемент не только по горизонтали, но и по вертикали. Для этого нужно использовать всего 3 свойства для родительского элемента: display: flex, justify-content: center и align-items: center.

  • Используем свойства Grid-модели.

    Относительно новая система верстки Grid-модель также предоставляет набор свойств для центрирования контента в блоке не только по горизонтали, но и по вертикали. Для этого нужно использовать всего 3 свойства для родительского элемента: display: grid, justify-items: center и align-items: center.
